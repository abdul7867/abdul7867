  <img alt="contribution snake" src="https://raw.githubusercontent.com/abdul7867/abdul7867/output/github-snake.svg"/>
</picture>

</div>

<br/>

<details>
<summary>&nbsp;▸ &nbsp;Activate the snake — one-time GitHub Actions setup</summary>

<br/>

Create `.github/workflows/snake.yml` inside this repo:

```yaml
name: Generate Snake

on:
  schedule:
    - cron: "0 */12 * * *"
  workflow_dispatch:

jobs:
  generate:
    runs-on: ubuntu-latest
    steps:
      - uses: Platane/snk/svg-only@v3
        with:
          github_user_name: ${{ github.repository_owner }}
          outputs: |
            dist/github-snake.svg
            dist/github-snake-dark.svg?palette=github-dark
      - uses: crazy-max/ghaction-github-pages@v3.1.0
        with:
          target_branch: output
          build_dir: dist
        env:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
```

Then go to **Actions → Generate Snake → Run workflow** to trigger it manually once.

</details>
